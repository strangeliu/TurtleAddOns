# AutoConfirm

## Overview

AutoConfirm is a World of Warcraft addon that automates confirmations for common popups, including accepting quests, completing quests, abandoning quests, replacing enchantments, and rolling on bind-on-pickup loot. It helps streamline gameplay by reducing repetitive clicks.

## Installation

1. Download the zip file off the turtle-wow branch of the repository.
2. Extract the zip file.
3. Remove the '-turtle-wow' suffix from the folder name.
4. Place the folder in your World of Warcraft `Interface/AddOns` directory.

## Usage

- In-game, type `/autoconfirm` or `/ac` to open the settings panel.
- Toggle automation for abandon quest, replace enchant, loot roll, and other popups.
- Changes are saved automatically.
