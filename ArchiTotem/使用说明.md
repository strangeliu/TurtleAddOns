# ArchiTotem 图腾大师V4 使用说明

## 🌟 插件简介

**ArchiTotem 是什么？**
ArchiTotem 是一个专门为魔兽世界萨满祭司设计的图腾管理插件。它能让您轻松管理所有图腾，就像使用一个智能的图腾工具箱一样！

**为什么选择 ArchiTotem？**
- 🎯 **简单易用**: 4个按钮管理所有图腾，告别复杂的图腾切换
- ⚡ **一键召唤**: 一次点击部署完整图腾组合
- 💾 **智能预设**: 内置PVE、PVP等场景预设
- 🔄 **图腾召回**: 节省法力值，重新布置图腾

**插件信息**
- **版本**: 1.6 (图腾大师V4)
- **适用**: 魔兽世界1.12.x (乌龟服定制版)
- **职业**: 仅萨满祭司可用
- **语言**: 支持中文、英文等多种语言

## 🚀 快速开始

### 第一步：安装插件
1. **下载**: 确保您已获得 ArchiTotem 插件文件
2. **安装**: 将 ArchiTotem 文件夹复制到 `Interface\AddOns\` 目录
3. **启用**: 进入游戏，在角色选择界面确认插件已启用
4. **使用**: 登录萨满祭司角色，插件会自动显示

### 第二步：认识界面
登录萨满祭司后，您会看到：
- **4个图腾按钮**: 分别代表大地、火焰、水之、空气图腾
- **一键图腾按钮**: 用于快速部署所有图腾
- **悬停展开**: 鼠标悬停在按钮上会显示更多图腾选项

### 第三步：开始使用
1. **施放图腾**: 左键点击任意图腾按钮施放图腾
2. **一键部署**: 左键点击一键图腾按钮，快速部署完整组合
3. **应用预设**: 输入 `/at preset pve` 使用PVE预设

**💡 新手提示**: 插件只在萨满祭司角色上工作，其他职业不会显示任何内容。

## 📖 基础操作教程

### 鼠标操作一览表
| 操作 | 功能 | 说明 |
|------|------|------|
| **左键点击** 图腾按钮 | 施放图腾 | 最基本的操作，释放选中的图腾 |
| **右键点击** 图腾按钮 | 跳过图腾 | 跳过的图腾会变红，一键召唤时不释放 |
| **Ctrl + 左键** 图腾按钮 | 图腾上移 | 提高图腾优先级，让它排在前面 |
| **Alt + 左键** 图腾按钮 | 图腾下移 | 降低图腾优先级，让它排在后面 |
| **Ctrl + 右键拖拽** 主按钮 | 移动界面 | 拖拽整个图腾条到合适位置 |
| **鼠标悬停** 图腾按钮 | 展开选项 | 显示该元素的所有可用图腾 |

### 一键功能操作
| 操作 | 功能 | 说明 |
|------|------|------|
| **左键点击** 一键按钮 | 一键召唤 | 依次释放所有图腾（跳过红色图腾） |
| **右键点击** 一键按钮 | 显示召回 | 显示召回按钮 |
| **左键点击** 召回按钮 | 召回图腾 | 召回所有图腾，回收部分法力值 |
| **右键点击** 召回按钮 | 隐藏召回 | 隐藏召回按钮 |

### 🎯 新手推荐操作流程

#### 第一次使用
1. **应用预设**: 输入 `/at preset pve` 使用PVE预设
2. **测试一键召唤**: 左键点击一键图腾按钮
3. **观察效果**: 看看游戏中是否成功部署了图腾

#### 日常使用
1. **选择场景**: 
   - 打怪升级用 `/at preset pve`
   - 打架PVP用 `/at preset pvp`
2. **一键部署**: 左键点击一键图腾按钮
3. **需要时调整**: 右键点击不需要的图腾设为跳过

#### 进阶使用
1. **自定义排序**: 用 Ctrl+左键 调整图腾优先级
2. **法力管理**: 使用召回功能节省法力值
3. **保存配置**: 用 `/at savepreset 我的配置` 保存个人设置

## 🎮 预设系统 - 新手最爱

### 什么是预设？
预设就是预先配置好的图腾组合，就像游戏中的天赋模板一样。插件内置了4种预设，适合不同的游戏场景。

### 内置预设说明

#### 🏹 PVE预设 - 打怪升级专用
```
/at preset pve
```
**图腾组合**:
- 🌍 **大地**: 大地之力图腾（+力量+敏捷，提高输出）
- 🔥 **火焰**: 火舌图腾（武器+火焰伤害，增加DPS）
- 💧 **水之**: 法力之泉图腾（回复法力，续航能力）
- 🌪️ **空气**: 风怒图腾（+攻击速度，提高输出）

**适用场景**: 刷怪、副本、任务

#### ⚔️ PVP预设 - 打架专用
```
/at preset pvp
```
**图腾组合**:
- 🌍 **大地**: 战栗图腾（解恐惧魅惑，PVP必备）
- 🔥 **火焰**: 灼热图腾（自动攻击敌人，防偷袭）
- 💧 **水之**: 治疗之泉图腾（持续治疗，提高生存）
- 🌪️ **空气**: 根基图腾（吸收法术，减少伤害）

**适用场景**: 战场、野外PVP、决斗

#### 🛡️ 抗性预设 - 抗伤害专用
```
/at preset resist
```
**图腾组合**:
- 🌍 **大地**: 石肤图腾（-物理伤害）
- 🔥 **火焰**: 抗寒图腾（+冰霜抗性）
- 💧 **水之**: 抗火图腾（+火焰抗性）
- 🌪️ **空气**: 自然抗性图腾（+自然抗性）

**适用场景**: 高难度副本、特殊环境

#### 🎯 自定义预设 - 平衡配置
```
/at preset custom
```
**图腾组合**:
- 🌍 **大地**: 大地之力图腾
- 🔥 **火焰**: 灼热图腾
- 💧 **水之**: 法力之泉图腾
- 🌪️ **空气**: 宁静之风图腾（-仇恨值）

**适用场景**: 团队副本、需要控制仇恨时

### 预设管理命令

#### 查看所有预设
```
/at listpresets
```
显示所有预设的详细内容，方便对比选择

#### 保存自定义预设
```
/at savepreset 我的配置
```
将当前的图腾配置保存为新的预设

**💡 新手建议**: 
- 刚开始建议使用 `pve` 预设
- 打PVP时切换到 `pvp` 预设
- 熟悉后可以创建自己的预设

## 🔧 常用命令速查

### 新手必备命令
```bash
/at preset pve          # 应用PVE预设（推荐新手）
/at preset pvp          # 应用PVP预设
/at callall             # 一键召唤所有图腾
/at recall              # 召回所有图腾
/at refresh             # 刷新界面（遇到问题时使用）
```

### 界面调整命令
```bash
/at scale 1.2           # 放大界面到1.2倍
/at direction up        # 图腾向上弹出
/at direction down      # 图腾向下弹出
```

### 图腾设置命令
```bash
/at set earth 3         # 显示3个大地图腾
/at set fire 2          # 显示2个火焰图腾
/at order earth fire water air  # 重置图腾顺序
```

### 功能开关命令
```bash
/at timers              # 开启/关闭冷却时间显示
/at tooltip             # 开启/关闭鼠标提示
/at showall             # 切换显示模式
```

## 💡 新手使用指南

### 🎯 第一次使用
1. **安装确认**: 登录萨满祭司，检查是否看到图腾按钮
2. **应用预设**: 输入 `/at preset pve` 
3. **测试功能**: 左键点击一键图腾按钮，看是否成功部署图腾
4. **熟悉界面**: 鼠标悬停在各个按钮上，了解功能

### 📋 日常使用流程
1. **根据场景选择预设**:
   - 打怪升级: `/at preset pve`
   - 打架PVP: `/at preset pvp`
   - 高难副本: `/at preset resist`

2. **部署图腾**: 左键点击一键图腾按钮

3. **调整设置** (可选):
   - 右键点击不需要的图腾设为跳过
   - 用 Ctrl+左键 调整图腾优先级

### 🔧 常见调整
- **界面太小**: `/at scale 1.5` 放大界面
- **位置不对**: Ctrl+右键拖拽移动位置
- **图腾顺序乱**: `/at order earth fire water air` 重置顺序
- **按钮不显示**: `/at refresh` 刷新界面

### 💰 法力值管理技巧
- **图腾召回**: 右键点击一键按钮→左键召回按钮，回收法力
- **跳过设置**: 右键点击暂时不需要的图腾，节省法力
- **预设切换**: 不同场景使用不同预设，避免浪费

### 🎮 进阶技巧
- **自定义预设**: 调整好图腾后用 `/at savepreset 我的配置` 保存
- **快捷键**: 熟悉 Ctrl+左键 和 Alt+左键 的排序功能
- **显示设置**: 用 `/at set earth 3` 设置显示更多图腾选项

## 🆘 问题解决

### 常见问题 & 解决方案

#### ❌ 插件不显示任何东西
**原因**: 角色不是萨满祭司或插件未正确加载
**解决**: 
1. 确认角色职业是萨满祭司
2. 检查插件文件是否在正确位置
3. 重新启动游戏

#### ❌ 图腾按钮不见了
**解决**: 输入 `/at refresh` 刷新界面

#### ❌ 一键召唤没反应
**可能原因**:
- 法力值不足
- 所有图腾都设为跳过
- 在战斗中无法施放某些图腾

**解决**: 
1. 检查法力值是否充足
2. 确认有图腾未被跳过（不是红色）
3. 脱离战斗后再试

#### ❌ 图腾位置混乱
**解决**: 输入 `/at order earth fire water air` 重置顺序

#### ❌ 界面太小看不清
**解决**: 输入 `/at scale 1.5` 放大界面
